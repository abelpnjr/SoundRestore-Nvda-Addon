Sound Restore NvdaAddon

Restores Sound After Random Disabling
This addon is simple to use.
after installing, by pressing nvda+escape key combination addon will atampt to re enable sound

Addon will do this actions:
1. Enabling sound on default sound device.
2. set volume of default sound device to 100.
3. removing the sound settings for spasific apps from system registry (requires re logon to system)

Notes
1. All Actions are done with default sound device.
2. this addon restores only system sound. It not useable to restore speech or hardvare problems.
